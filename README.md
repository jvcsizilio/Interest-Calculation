## Description

Application in Heroku using:

- Nodejs
- Angular 1.x
- bootstrap
- bower
- gulp


## Installation

```
npm install
bower install
gulp

```

## Start Nodejs Express 4 server
```
npm start
```
Or if you have a Heroku account

```
heroku local
```
